# -- COMMANDS --
- NUMPAD `+` -> Zoom in
- NUMPAD `-`-> Zoom out
- `ARROWS` -> for direction


## REQUIREMENTS
> `Pyopengl` pip install PyOpenGL PyOpenGL_accelerate
 
> `Numpy` pip install numpy

> `Pygame` pip install pygame
